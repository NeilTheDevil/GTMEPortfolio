# Portfolio Creation Skill (Royal GTM Edition)

This skill generates a high-end, editorial portfolio for GTM Engineers, specifically optimized for the 'Royal GTM' aesthetic (Navy, Gold, and Warm Cream).

## Design Philosophy

- **Layout**: Single-column, linear flow. 
- **Typography**: Extreme scale variations (headers up to 9rem, subheaders at 1.5rem). Use 'Inter'.
- **Palette**: Royal GTM (#FCFBF7 background, #000C24 text, #B8860B gold accents).
- **Hero Image**: Full-color portrait, high precision.

## Core Sections

1. **Editorial Hero**: Full-color portrait + oversized headline. No grayscale filters.
2. **Specialized Proof**: 
   - GTM Data Engineering (Loom Grid)
   - Lead Generation at Scale (10k+ Stat & Tool Badges)
   - Operations & Automation (Highlight Boxes & Node Visuals)
3. **Linear Work**: Large, full-width headers for major case studies.
4. **Community Engagement**: 2-column layout with image gallery on left and highlights on right.
5. **About Me**: Narrative with secondary portrait area.
6. **High-Impact CTA**: Blue/Gold emphasis on 'revenue' and stylized 'Lets Talk' button.

## Execution Steps for the Agent

1. **Content Extraction**: Extract specific GTM credentials (Leads generated, Tools used).
2. **Palette Mapping**: Ensure navy matches the user's primary professional attire in their portrait.
3. **Template Application**: Use the files in `resources/`.
4. **Automation Details**: Include custom Claude Code/n8n/LLM workflow descriptions.
